Array Sorting Logic & Validation (C++)
Цей проєкт реалізує алгоритм перевірки можливості впорядкування масиву чисел за специфічними правилами інкременту. Робота демонструє навички роботи з динамічними структурами даних та оптимізацію перебору елементів масиву.

Технології:
C++.
Контейнер std::vector — використаний для гнучкої роботи з послідовностями даних, розмір яких визначається під час виконання програми.
Бібліотека <windows.h> — для забезпечення коректної підтримки кириличного тексту в консольному інтерфейсі.

Що робить ця програма:
Динамічне введення даних: Програма приймає кількість елементів $n$ та формує відповідний масив чисел.
Алгоритм валідації: Проходить крізь масив та перевіряє, чи не є поточне число критично меншим за попереднє (різниця не повинна перевищувати 1).
Умовне редагування: Якщо поточне число рівне попередньому мінус один, програма автоматично збільшує його на одиницю для спроби стабілізації послідовності.
Логіка швидкого виходу: Використовує прапорець possible та оператор break для миттєвого завершення аналізу, якщо умову неможливо виконати, що економить ресурси.
